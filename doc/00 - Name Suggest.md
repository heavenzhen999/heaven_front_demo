# document Naming

> 文件命名规范

## 文件命名

文件命名统一使用小写字母。使用中划线进行分割。

## 图片命名

命名规范统一使用小写字母。

根据"用处-修饰词-大小"来进行命名。

例如 "logo-red-32px.png"来命名文件
